## Releases

[GhostBSD 20.11.28 Release Announcement](https://ghostbsd.org/20.11.28_release_announcement?utm_source=bsdweekly): This release comes with a new live system that leverages ZFS, compression, and replication first introduced in FuryBSD by Joe Maloney. The 20.11.28 release contains numerous improvements, including OS fixes for linuxulator to improve Linux Steam performance, an updated kernel, and GhostBSD userland updates. Userland updates include a MATE desktop upgrade to version 1.24.1, Software Station performance improvements, and numerous application updates.

[TrueNAS 12.0-U1 is Scheduled for early December](https://www.ixsystems.com/blog/truenas-12-u1-is-scheduled/?utm_source=bsdweekly): In just four weeks, more than 20,000 systems have been upgraded to TrueNAS 12.0. TrueNAS 12.0-U1 has entered development code-freeze and is now entering its final QA cycle for availability in early December.

## News

[FreeBSD Merges WireGuard Support](https://www.phoronix.com/scan.php?page=news_item&px=FreeBSD-WireGuard-Lands&utm_source=bsdweekly): The momentum of WireGuard continues with FreeBSD now having mainlined their port of this open-source secure VPN tunnel. For FreeBSD 13 there is now the import of the WireGuard kernel module. This follows OpenBSD adding WireGuard earlier in the year.

[macOS to FreeBSD migration a.k.a why I left macOS](https://antranigv.am/weblog_en/posts/macos_to_freebsd/?utm_source=bsdweekly): This is not a technical documentation for how to migrate from macOS to FreeBSD. This is a high-level for why to migrate from macOS to FreeBSD.

[re(4) testers needed for DragonFly](https://www.dragonflydigest.com/2020/11/25/25182.html?utm_source=bsdweekly): If you have a Realtek network card supported by the re(4) driver, Sepherosa Ziehau has a new driver for you to test.

[OpenZFS 2.0 Released With Unified FreeBSD/Linux Support, Many New Features](https://www.phoronix.com/scan.php?page=news_item&px=OpenZFS-2.0-Released&utm_source=bsdweekly): OpenZFS 2.0 marks a major step forward for open-source ZFS file-system support for what started out as ZFSOnLinux but is now OpenZFS with unified FreeBSD and Linux support (macOS support is still being pursued as well) and this release also bringing many new features.
